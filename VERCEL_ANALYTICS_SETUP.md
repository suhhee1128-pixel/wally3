# Vercel Analytics 활성화 가이드

## 📊 Web Analytics 활성화

### 단계별 방법:

1. **Vercel Dashboard 접속**
   - https://vercel.com/yeonwoos-projects-d44e7542/wally3

2. **Settings로 이동**
   - 왼쪽 메뉴에서 **"Settings"** 클릭

3. **Analytics 섹션 찾기**
   - Settings 페이지에서 **"Analytics"** 섹션 찾기
   - 또는 상단 검색창에 "Analytics" 입력

4. **Web Analytics 활성화**
   - **"Enable Web Analytics"** 버튼 클릭
   - 또는 토글 스위치를 켜기

5. **확인**
   - 활성화되면 "Web Analytics is enabled" 메시지 표시
   - Dashboard 메인 페이지에서 Analytics 데이터 확인 가능

## ⚡ Speed Insights 활성화

### 단계별 방법:

1. **Settings → Speed Insights로 이동**
   - Settings 페이지에서 **"Speed Insights"** 섹션 찾기

2. **Speed Insights 활성화**
   - **"Enable Speed Insights"** 버튼 클릭
   - 또는 토글 스위치를 켜기

3. **확인**
   - 활성화되면 "Speed Insights is enabled" 메시지 표시

## 📈 데이터 확인 방법

### Analytics 데이터 보기:

1. **Dashboard 메인 페이지**
   - 프로젝트 선택 후 메인 페이지에서 Analytics 카드 확인

2. **Analytics 탭**
   - 왼쪽 메뉴에서 **"Analytics"** 클릭
   - 페이지뷰, 고유 방문자, 인기 페이지 등 확인

3. **Speed Insights 탭**
   - 왼쪽 메뉴에서 **"Speed Insights"** 클릭
   - 성능 메트릭 확인
